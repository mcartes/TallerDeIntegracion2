Bienvenidos a nuestro proyecto

LeFrogé es un proyecto grupal creado para el ramo Taller de integración II. El proyecto consiste en crear un entorno digital estructurado en base al proceso científico para crear, escribir, editar, guardar y exportar artículos científicos. El investigador tiene la posibilidad de interactuar con los diversos elementos constitutivos de un proceso de investigación, los cuales se integran para dar el formato y las orientaciones básicas del diseño metodológico seleccionado. De esta forma el proceso de elaboración del documento científico es más amigable y accesible para personas menos experimentadas y optimiza el proceso de aquellos usuarios expertos.
