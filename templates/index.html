Bienvenidos a nuestro proyecto

Le_Frogé es un proyecto grupal creado para el ramo Taller de integración II. El proyecto consiste en crear un entorno digital estructurado en base al proceso científico para crear, escribir, editar, guardar y exportar artículos científicos. El investigador tiene la posibilidad de interactuar con los diversos elementos constitutivos de un proceso de investigación, los cuales se integran para dar el formato y las orientaciones básicas del diseño metodológico seleccionado. De esta forma el proceso de elaboración del documento científico es más amigable y accesible para personas menos experimentadas y optimiza el proceso de aquellos usuarios expertos.


Funcionalidades

La página cuenta con diversas funcionalidades tales como:

  • Subir tus propios archivos.
  • Crear tu propio documento desde cero.
  • Revisar las categorías de los archivos subidos.
  • Poder editar tus propios documentos.
  • Descargar tu documento en diversos formatos(PDF, ODT, JSON).

Tecnologías empleadas

Para la creación de nuestro proyecto hemos utilizado distintas tecnologías y lenguajes.

  • Python, Flask y MongoDB.
  • HTML, CSS y JavaScript.
  • Bootstrap, JQuery y Froala.
